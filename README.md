# Store Rating Application

A full-stack web application where users can browse stores and submit ratings.

## Tech Stack

* Java
* Spring Boot
* Spring Security
* PostgreSQL
* React.js
* Bootstrap
* JWT Authentication

## Features

### Admin

* Manage Users
* Manage Stores
* Assign Store Owners
* View Dashboard Statistics

### Store Owner

* View Store Details
* View Ratings and Average Rating

### User

* Register Account
* Login
* Browse Stores
* Submit and Update Ratings
